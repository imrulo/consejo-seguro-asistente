# ConsejoSeguro Backend

Este directorio contendrá el backend de la plataforma, construido con Nest.js (Node.js), PostgreSQL, Redis y Supabase.

## Estructura esperada
- API REST/GraphQL
- Integración con IA (OpenAI, Hugging Face)
- Autenticación (Firebase Auth, OAuth2)
- Sincronización offline
